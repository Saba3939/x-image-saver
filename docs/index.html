なぜ X Image Catcher？

Xのタイムラインは流れが速い。気になった画像を後から探すのは大変です。

⚡
スクロールするだけ
特別な操作は一切不要。x.com を普通に使っているだけで、自動的に画像がキャプチャされます。
🔒
完全ローカル保存
データは外部サーバーには送信されず、あなたのブラウザのみに保存されます。プライバシーは完全に守られます。
🔍
@handle で絞り込み
「あのアカウントの画像だけ見たい」という時に、投稿者のハンドル名で瞬時に絞り込めます。
📌
最大1,000件保存
最新1,000件まで自動で蓄積。古いものから順に削除されるので、ストレージを圧迫しません。
🚫
広告は自動スキップ
プロモーション投稿（広告）の画像は自動的に除外されます。純粋なタイムラインの画像だけが保存されます。
⌨️
キーボードショートカット
Alt+X を押すだけでポップアップが開きます。マウスを動かさずにギャラリーにアクセスできます。

使い方は超シンプル

たった3ステップで始められます

1

Chrome に拡張機能を追加する

「Chrome に追加」ボタンをクリックして、拡張機能をインストールします。

2

x.com を開いてスクロールする

普段通りタイムラインを眺めるだけ。画像が表示されると自動的にキャプチャされます。

3

アイコンをクリックしてギャラリーを開く

ブラウザ右上の拡張機能アイコン（または Alt+X）でポップアップを開き、保存した画像を一覧できます。

🔐

プライバシーを最優先に設計

X Image Catcher はあなたのデータを一切外部に送信しません。 画像URLとツイートのメタデータはすべてあなたのブラウザ内にのみ保存されます。
