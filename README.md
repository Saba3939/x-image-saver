# X Image Catcher

X（旧Twitter）のタイムラインをスクロールするだけで、画像を自動キャプチャして保存するChrome拡張機能です。

## 機能

- **自動キャプチャ** — タイムラインをスクロール中に表示された画像を自動保存
- **ギャラリー表示** — 最大1,000枚を2カラムグリッドで新しい順に表示
- **検索・フィルター** — @ハンドルや表示名でリアルタイム絞り込み
- **ツイートへジャンプ** — 画像をクリックすると元ツイートを新しいタブで開く
- **投稿者情報の表示** — アバター・@ハンドルをサムネイルにオーバーレイ表示
- **広告の自動除外** — プロモーション投稿の画像はキャプチャしない
- **削除機能** — 1枚ずつ、またはギャラリー全件を削除
- **未読バッジ** — 新たにキャプチャされた枚数を拡張機能アイコンに表示
- **キーボードショートカット** — `Alt+X` でポップアップを即時表示

## インストール方法

### Chrome Web Store（準備中）

Chrome Web Store からインストールできます（公開準備中）。

### 手動インストール（開発者向け）

1. このリポジトリをクローン、またはZIPでダウンロードして解凍する
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このフォルダを選択する

## 使い方

1. X（x.com）を開き、タイムラインをスクロールする
2. 表示された画像が自動的に保存される
3. 拡張機能アイコンをクリック（または `Alt+X`）してギャラリーを開く
4. 検索ボックスに @ハンドルや名前を入力して絞り込む
5. 画像をクリックすると元ツイートが開く

## データの取り扱い

- すべてのデータは **ローカル（`chrome.storage.local`）にのみ保存** されます
- 外部サーバーへの送信は一切行いません
- 保存上限は **1,000枚**（上限超過時は古いものから自動削除）

## 必要な権限

| 権限 | 用途 |
|------|------|
| `storage` | 画像URLとメタデータのローカル保存 |
| `host_permissions` (x.com, twitter.com) | コンテンツスクリプトの挿入 |

## 動作環境

- Google Chrome（Manifest V3 対応）
- x.com / twitter.com
